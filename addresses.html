Contact Form Address Configuration

Contact form uses a set of alias (the %Aliases variable) to allow email to be sent. Each alias has two parts: a public name, and a private emal address. The address is never shown on the contact form page or in the page source.

Default Configuration

my %Aliases = (
	'administrator',&safeHeader($ENV{'SERVER_ADMIN'}),
	#'webmaster','webmaster@yoursite.tld',
	#'postmaster','postmaster@yoursite.tld',
);
The two lines that start with # are commented out. In this configuration the form will send email only to the web server administrator (as defined in the web server configuration). It is almost certain that you want to change this.

Just One Person

my %Aliases = (
	'John Doe','johndoe@example.com',
);
In this configuration, only John Doe can get email. Users will not be given a choice to send email to other people.

Multiple People

my %Aliases = (
	'John Doe','johndoe@example.com',
	'Jane Doe','janedoe@example.com',
	'The Webmaster','webmaster@example.com',
	'Sales Department','sales@example.com',
);
In this configuration, the form can send email to John, Jane, Sales, or Webmaster. Users of the will see a drop down menu from which they can choose one of these four options.
